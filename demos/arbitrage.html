How to Run This Demo

Requirements: Python 3.8+, pip package manager

Installation:

pip install streamlit requests pandas numpy matplotlib

Launch the Application:

cd demos
streamlit run arbitrage.py

Note: The application will open in your default web browser at http://localhost:8501

Online Deployment:

For cloud deployment (optional):

streamlit run arbitrage.py --server.port 8501 --server.address 0.0.0.0
